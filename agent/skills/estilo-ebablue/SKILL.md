# Skill: Estilo y Marca – EBABlue Handmade

Marca: EBABlue Handmade

## Identidad
Bisutería artesanal femenina, delicada y elegante.
Inspiración: mariposa, ligereza, detalles hechos a mano, feminidad moderna.

## Cuándo usar este Skill
- Diseño de landing
- Creación de UI
- Publicaciones web
- Copies de productos
- Botones, mensajes y descripciones

## Regla número 1
No improvisar colores ni tono. Usar exclusivamente los definidos en los recursos.

## Dónde mirar según la tarea
- Estilo visual → recursos/estilo-visual.json
- Tono de marca → recursos/guia-de-textos.md
- Reglas técnicas → recursos/reglas-tecnicas.md

## Checklist antes de entregar
1) ¿Transmite delicadeza y elegancia?
2) ¿Los colores respetan la paleta oficial?
3) ¿El tono es femenino y cercano?
4) ¿No parece una tienda genérica?
5) ¿Hay aire y limpieza visual?
